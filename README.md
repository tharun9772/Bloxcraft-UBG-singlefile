# Bloxcraft UBG
A single-file version of Bloxcraft UBG.

May not get updated as fast as the main version.
Everything is empty, this is a beta project.

# -----
**How it works:**

It will rewrite all the html and replace routes such as `/example.png` to `{$base}/example.png`

The base would probbaly be the jsdelivr version @ https://github.com/tharun9772/Bloxcraft-UBG

The proxy will be unstable.
